# Exomind Albums Room MVVM Koin

Exomind albums est une application pour afficher une liste des utilisateurs avec leurs photos et albums

### Features
- Kotlin language
- MVVM Architecture
- Retrofit
- Coroutines
- LiveData
- Glide
- Room
- Rxjava
- LifeCycle
